## 安装

> 组件涉及`jsx`操作， 可以[点击这里学习](https://cn.vuejs.org/v2/guide/render-function.html)

**vue支持 jsx写法**
```shell
npm i babel-plugin-syntax-jsx babel-plugin-transform-vue-jsx --save-dev

# 在.babelrc文件中加入
"plugins": ["transform-vue-jsx"]

```

**使用 npm 的安装**

```shell
npm i @qtt/q-form -S
```

**使用 yarn 的安装**

```shell
yarn add @qtt/q-form
```
